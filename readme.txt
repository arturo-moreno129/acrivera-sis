*********Activar la extensión ZipArchive en PHP para el uso de librería PhpSpreadsheet ******************
Paso 1: Abre el archivo de configuración de PHP, php.ini. Este archivo normalmente se encuentra en la carpeta de instalación de XAMPP (C:\xampp\php\php.ini).

Paso 2: Busca la siguiente línea:
;extension=zip
Asegúrate de quitar el punto y coma al inicio de la línea, dejando esto:
extension=zip
